Voice Controlled Wheelchair

Developing a voice-controlled chair that obeys the orders of a specific user, using a VR shield, to perform a variety of tasks. Moreover, there is an ultrasonic sensor to prevent collapsing with nearby objects.

The project passed through these steps:-

- Programming the VR shield to recognize (Forward, Backward, Right, Left, and Stop) commands.

- Interfacing the VR with the Arduino ship and programming it to convert the aforementioned commands into actions and send it to the L298 module using Arduino C language.

- Designing a protective box for the Arduino ship and the VR shield using Corel Draw X6.

- Printing the box design on acrylic sheets using the laser cutter machine.

- Programming the Ultra Sonic sensor to cancel any order if there is an object in its range.

- Fixing the components to the metal base using copper spacers.

According to DC motor laws, the relation between the current and the weight is I = N*µ*R/KT

Where (I) is the current in amperes, (N) is the normal reaction (mass*gravity), and (R) is the radius.

We set different times for rotations (Right & Left) commands in the Arduino code; then, we measured the angles associated with it. Thus, there is an inverse relationship between the Coefficient of friction (µ) and the angle of rotation. In addition, our objective is to stop the chair when there is an obstacle at a distance of 20 cm or less. Thus, we programmed the ultrasonic to do. The VR shield (voice recognition shield) is the cornerstone of our chair. It turns the human voice into representable waves and data using an analog-to-digital module and then records it with all the properties associated with it.

The main purpose of communication is to provide an easier life for the people where they use the minimum effort to achieve things which would have required tremendous effort using the conventional ways.
